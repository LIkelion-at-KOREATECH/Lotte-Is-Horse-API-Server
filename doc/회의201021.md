### 롯데 api를 받지 못하는 문제는 어떻게 해결?
모델 -> api -> 자바스크립트에서 json으로 받아서 처리

#### 모델
상품, 지점, 판매하다

#### api
Django REST framework

### 자바스크립트
api 연결
직렬화
